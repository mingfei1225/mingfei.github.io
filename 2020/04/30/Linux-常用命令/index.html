Linux 常用命令

ls 命令 [选项] [参数]
-a 显示所有文件，包括隐藏文件
-l 显示详细信息
-d 查看目录属性
-h 人性化显示文件大小
-i 显示inode
mkdir -p
ctrl + l 清屏（xshell）
cd ~ 进入当前用户家目录
cd
cd - 进入上次目录
cd .. 进入上一级目录
cd . 进入当前目录

命令补全 tab键单击或者双击

pwd （print working directory）

rmdir

rm 删除文件或目录
-r 删除目录
-f 强制删除

cp [选项] [源文件] [目标位置]
-r 复制目录
-p 连带文件属性复制
-d 若源文件是链接文件，则复制链接属性
-a 相当于 -pdr

mv 剪切或改名
mv [源文件或者目录] [目标目录]

ln
-s 软连接（一定要写绝对路径）
了解软连接和硬连接区别

文件搜索命令
locate 文件名 搜索速度快，搜索的mlocate 数据库，一天一更新，实时性不好，可以用updatedb手动更新mlocate。
命令的搜索命令：
whereis 命令名 搜索命令所在路径以及帮助文档所在位置
-b 只查找可执行文件
-m 只查找帮助文件
which 命令 和whereis基本一样，还能看到别名所在位置。
find 功能强大，搜索所有文件名，慢
find [搜索范围] [搜索条件] 搜索文件
示例：find / -name install.log
支持通配符：
* 匹配任意内容
？ 匹配任意一个字符
[] 匹配任意一个中括号内的字符
支持条件
-name 按文件名搜索
-iname 按文件名搜索，不区分大小写
-user 按所有者搜索
-nouser 查找没有所有者的文件（由内核产生，或者外部传入）
-mtime 查找符合修改文件时间的文件 单位默认是天
示例：find /var/log/ -mtime +10 #查找10天前修改的文件
-10 10天内修改的文件
10 10天当天修改的文件
+10 10天前修改的文件
-size 查找符合文件大小的文件
示例：find /root -size +25k #查找root目录下大于25k的文件
25k 参考mtime
size 类型：k，M（区分大小写）
-inum 根据inode查找
还支持多条件搜索
-a and
-o or
-exec 命令 {} ; #复合命令
grep 搜索文件字符串
grep [选项] 字符串 文件名 #在文件中匹配符合条件的字符串
-i 忽略大小写
-v 排除指定字符串

帮助命令
man
命令 –help
help 专门获取内部（shell自带）命令

压缩命令
zip [目标位置] [源文件位置]
gzip gunzip
bzip2 [源文件] 压缩为.bz2，不保留源文件
bzip2 -k 源文件 压缩之后保留源文件
bzip2 -d 解压缩
bunzip2 解压缩
tar -cvf 打包文件名 源文件
-c 打包
-x 解包
-z 压缩为.tar.gz格式
-j 压缩为.tar.bz2格式
-v 显示过程
-f 指定打包后的文件名
示例： tar -cvf mingfei.tar mingfei
tar -xvf mingfei.tar

关机与重启命令
shutdown [选项] 时间
-c 取消前一个关机命令
-h 关机
-r 重启
示例：shutdown -r 05:30 & （&的意思是将命令后台执行）
halt 以下三个命令都不太安全
poweroff
init 0
reboot 较安全
init 6
0 关机
1 单用户（类型windows安全模式）
2 不完全多用户，不含NFS服务
3 完全多用户（普通当前界面）
4 未分配
5 图形界面
6 重启
可用runlevel命令查看当前系统运行级别
logout 退出命令（每次管理完，一定要正确退出）

挂载命令
光盘：
mount 挂载
示例： mount -o remount,exec /home/
mount -t iso9660 /dev/sr0 /mnt/cdrom
mount /dev/sr0 /mnt/cdrom
umount 卸载
示例： umount /mnt/cdrom/ #注意要退出cdrom
U盘：
fdisk -l 查看
mount -t vfat /dev/sdb1 /mnt/usb/

用户登录查看
w 查看登录了哪些用户
who 查看登录用户概要信息
last 查看当前登录和过去登录的用户信息（包含重启时间）
读取的是/var/log/wtmp文件
lastlog
读取的是/var/log/lastlog文件

shell基础
echo
echo -e “\e[1;31m嫁人就要嫁凤姐\e[0m” #\e[1; 开启颜色 \e[0m 关闭颜色
vim hello.sh
#!/bin/bash
# The first program

    echo -e "e\[1;34m天上掉下个林妹妹e\[0m"
bash hello.sh
chmod 755 hello.sh
./hello.sh

别名
alias ls=’lis –color=never ‘
alias vi=’vim’
unalias vi
优先级：
1、执行绝对路径或者相对路径的命令
2、执行别名
3、执行Bash的内部命令
4、执行按照$PATH环境变量定义的目录查找顺序找到的第一个命令

常用快捷键
ctrl+c 强制终止当前命令
ctrl+l 清屏
ctrl+a 光标移动到命令行首
ctrl+e 光标移动到命令行尾
ctrl+u 从光标所在位置删除到行首
ctrl+z 把命令放入后台
ctrl+r 在历史命令中搜索

输出重定向
命令 > 文件 2>&1 以覆盖的方式，把正确输出和错误输出都保存到同一个文件中
命令 >> 文件 2>&1 以追加的方式。。。
命令 &> 文件 以覆盖的方式。。。
命令 &> 文件 以追加的方式。。。
命令 >> 文件1 2>>文件2 把正确的输出追加到文件1，错误的输出追加文件2。
